# File Management Utility

A Python-based file management system that helps users create, backup, organize, and log files automatically.

## Features

- **Project Initialization**: Creates and manages the StudentFiles directory
- **File Creation**: Generates dated files with student names
- **File Information**: Displays file contents, size, and modification date
- **Backup System**: Creates backup copies and archives them
- **Logging**: Maintains an activity log with timestamps
- **File Operations**: Supports file deletion and directory listing

## Requirements

- Python 3.6+
- No external dependencies (uses only built-in modules)
